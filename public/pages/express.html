Definition:

from expressjs.com:


'Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.
With a myriad of HTTP utility methods and middleware at your disposal, creating a robust API is quick and easy.
Express provides a thin layer of fundamental web application features, without obscuring Node.js features that you know and love. '


from developer.mozilla.org:


'Express is the most popular Node web framework, and is the underlying library for a number of other popular Node web frameworks. It provides mechanisms to:
- Write handlers for requests with different HTTP verbs at different URL paths (routes).
- Integrate with "view" rendering engines in order to generate responses by inserting data into templates.
- Set common web application settings like the port to use for connecting, and the location of templates that are used for rendering the response.
- Add additional request processing "middleware" at any point within the request handling pipeline.'


How to use express:

First you must add express to your project using npm
npm install express Or alternatively add express in your package.json file

Then import express into your main javascript file: const express = require("express");
const app = express();

You are now ready to use all available express functions using the "app" variable.
in the "Basic Webserver page, some of those functions are demonstrated.
