How to setup a basic webserver using Node.js, npm, and express


Step 1:

The first step is to create a directory to initiate your project in, then go to that directory
mkdir webServer
cd webServer


Step: 2

Then initiate a project with npm and install express
npm init
npm install express


Step: 3

Now set up your main javascript file:
First import express and initiate it in a variable: const express = require("express");
const app = express();
Then invoke the "listen" function to access a port to host your server: app.listen(3000);
Finally add the function to handle a simple api request: app.get('/', (req, res) => {
res.send('Hello World!');
});


Step: 4

You now have a functioning webserver at "localhost:3000/"
You can add "app.post", "app.put! and so on, to create a fully functional CRUD RestAPI webserver! (if you add a database)
