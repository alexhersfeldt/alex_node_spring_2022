Definition:

from nodejs.org:


'npm is two things: first and foremost, it is an online repository for the publishing of open-source Node.js projects; second, it is a command-line utility for interacting with said repository that aids in package installation, version management, and dependency management.'


from npmjs.com:


' npm is the package manager for Node.js. It was created in 2009 as an open source project to help JavaScript developers easily share packaged modules of code.

The npm Registry is a public collection of packages of open-source code for Node.js, front-end web apps, mobile apps, robots, routers, and countless other needs of the JavaScript community.

npm is the command line client that allows developers to install and publish those packages.'


How to use npm:

Node.js is installed by default with npm

To initiate npm in your project run the following command:
npm init This will create the package.json file, which is used to organize the repositories you are using.

Simply use the npm keyword in front of a repository name ( and evt. setup keywords) and the repository will be installed. Repository library can be found at npmjs.com

Example:
npm i {repository name} This will install a repository.

Alternatively add the repository to your package.json file
